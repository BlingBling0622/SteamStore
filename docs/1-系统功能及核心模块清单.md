# 系统功能及核心模块清单

## 一、系统概述

本系统旨在**提供一个Steam风格的游戏商城平台，实现游戏商品的浏览、购买与个人游戏库管理**，基于 **SpringBoot、Spring Security、Spring Data JPA、MySQL、Thymeleaf** 技术搭建，具备**高稳定性、数据持久化可靠、Session安全管理、多语言国际化支持**等特点，可满足**游戏爱好者、个人用户**在**游戏商品浏览、在线购买、个人库管理**方面的需求。

## 二、功能清单

| 功能模块 | 功能名称 | 功能描述 | 功能特点 |
|---------|---------|---------|---------|
| 用户系统 | 用户注册 | 用户通过用户名、邮箱、密码创建账号 | 密码BCrypt加密，用户名/邮箱唯一性校验，表单验证 |
| 用户系统 | 用户登录 | 用户凭用户名和密码登录系统 | Spring Security认证，Session持久化，登录失败提示 |
| 用户系统 | 用户登出 | 用户安全退出系统 | 清除Session，CSRF保护，重定向到登录页 |
| 商品浏览 | 首页展示 | 展示精选游戏和全部商品 | 精选游戏大图展示，分类网格展示，认证状态感知导航 |
| 商品浏览 | 商品列表 | 展示所有130+款游戏商品 | 列表视图，显示标签/评价/价格，折扣显示，多语言价格 |
| 商品浏览 | 商品详情 | 展示单个游戏的详细信息 | 图片、描述、开发商、评价、标签、购买按钮联动状态 |
| 购物车 | 添加商品 | 将游戏加入购物车 | 防重复添加，已登录才可操作，实时计数显示 |
| 购物车 | 查看购物车 | 展示购物车内所有商品及总价 | 折扣价格计算，多商品汇总，支持移除 |
| 购物车 | 移除商品 | 从购物车中删除指定商品 | 按商品ID精确删除，操作后自动刷新 |
| 订单系统 | 结账 | 将购物车商品一键下单 | 事务保护，自动清空购物车，生成订单记录 |
| 订单系统 | 订单记录 | 记录用户每次购买的订单 | 多对多关联商品，记录购买时间和总价，永久保存 |
| 个人库 | 游戏库展示 | 展示用户所有已购买游戏 | 购买游戏统一展示，游戏封面图，游戏名显示 |
| 个人库 | 已拥有标记 | 商店内标记用户已拥有的游戏 | 商品列表和详情页均显示"已在库中"，防止重复购买 |
| 国际化 | 多语言支持 | 支持8种语言界面切换 | 英/中/日/韩/法/德/俄/西语，URL参数切换，即时生效 |
| 国际化 | 多货币支持 | 根据语言自动切换货币和价格 | 支持USD/CNY/JPY/KRW/EUR/RUB，汇率自动换算 |

## 三、核心模块说明

### 3.1 用户认证模块
- **技术**: Spring Security 6
- **功能**: 注册、登录、登出、Session管理
- **安全**: BCrypt密码加密，CSRF保护，Session固定攻击防护
- **相关文件**: `SecurityConfig.java`, `AuthController.java`, `UserService.java`, `CustomUserDetailsService.java`

### 3.2 商品管理模块
- **技术**: Spring Data JPA + MySQL
- **功能**: 商品CRUD、精选商品查询、按ID查询
- **数据**: 130+款游戏，包含名称/价格/折扣/图片/标签/评价/开发商
- **相关文件**: `Product.java`, `ProductRepository.java`, `StoreService.java`, `StoreController.java`

### 3.3 购物车模块
- **技术**: JPA事务管理
- **功能**: 添加/查看/删除购物车商品，计算总价
- **约束**: 用户-商品唯一约束，防止重复添加
- **相关文件**: `CartItem.java`, `CartRepository.java`, `StoreService.java`

### 3.4 订单模块
- **技术**: JPA多对多关联，`@Transactional`事务
- **功能**: 创建订单、记录购买历史、查询已拥有游戏
- **流程**: 购物车 → 创建订单 → 关联商品 → 清空购物车（原子操作）
- **相关文件**: `Order.java`, `OrderRepository.java`, `StoreService.java`

### 3.5 国际化模块
- **技术**: Spring MessageSource + Thymeleaf i18n
- **功能**: 8种语言界面，8种货币自动换算
- **机制**: URL参数`?lang=xx`切换，`GlobalModelAdvice`全局注入货币信息
- **相关文件**: `messages_*.properties`, `GlobalModelAdvice.java`, `WebConfig.java`

### 3.6 数据初始化模块
- **技术**: Spring `CommandLineRunner`
- **功能**: 应用启动时自动初始化测试数据
- **内容**: 3个测试用户，130+款游戏商品
- **相关文件**: `DataInitializer.java`, `TestDataSeeder.java`

## 四、系统技术架构

```
┌─────────────────────────────────────────┐
│              前端（Thymeleaf）            │
│  index.html / store.html / dashboard.html│
│  cart.html / login.html / register.html  │
└──────────────────┬──────────────────────┘
                   │ HTTP请求
┌──────────────────▼──────────────────────┐
│            控制器层（Controller）          │
│  HomeController / StoreController        │
│  AuthController / DashboardController    │
└──────────────────┬──────────────────────┘
                   │ 调用服务
┌──────────────────▼──────────────────────┐
│             服务层（Service）              │
│  StoreService / UserService              │
│  CustomUserDetailsService                │
└──────────────────┬──────────────────────┘
                   │ 数据访问
┌──────────────────▼──────────────────────┐
│          数据访问层（Repository）           │
│  ProductRepository / CartRepository      │
│  OrderRepository / UserRepository        │
└──────────────────┬──────────────────────┘
                   │ SQL
┌──────────────────▼──────────────────────┐
│            数据库（MySQL 8.0）             │
│  steam_store 数据库                       │
│  users / products / cart_items / orders  │
└─────────────────────────────────────────┘
```

## 五、系统特点

- **安全性**: Spring Security全面保护，BCrypt加密，CSRF防护
- **完整性**: 前后端完整实现，数据库持久化
- **可用性**: 130+款游戏数据，多语言支持，响应式设计
- **可靠性**: 事务保护关键操作，防止数据不一致
- **扩展性**: MVC分层架构，便于功能扩展
